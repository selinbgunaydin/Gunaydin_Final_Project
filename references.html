Technical Frameworks

This portfolio-quality mock-up was developed using modern web technologies to ensure responsiveness and accessibility:

  • Accessibility: Optimized for WAVE compliance (Web Accessibility Evaluation Tool).
  • Development: Built with HTML5 and custom CSS3 styles.

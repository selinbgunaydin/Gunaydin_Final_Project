# Macon County NC - Website Redesign 

**Student:** Selin Gunaydin  
**Course:** WEB210 - Web Design  
**Project Date:** April 2026  

## Project Overview
This project is a 8-page website redesign for Macon County, NC. It focuses on accessibility (WAVE compliant), responsive web design, and a user-centered interface.

## Technologies Used
* **HTML5 & CSS3** (Custom styles)
* **Bootstrap v5.2.3** (Grid system & Components)
* **WAVE Tool** (Accessibility auditing)
